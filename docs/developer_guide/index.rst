Overview
--------

The Aerial Framework provides a reference architecture for building and running GPU-accelerated
signal processing workloads. This guide covers container setup, build configuration,
testing, C++ and Python development workflows, dependencies, and a guide for running the
real-time applications so you can develop your own pipelines.
