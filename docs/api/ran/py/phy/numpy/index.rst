NumPy PHY
=========

Pure NumPy reference implementations of physical layer algorithms.

Overview
--------

The NumPy PHY module provides reference implementations using pure NumPy:

* **PUSCH** (:doc:`pusch`) - Physical Uplink Shared Channel receiver algorithms
* **Utils** (:doc:`utils`) - Signal processing utilities (modulation, sequences)
